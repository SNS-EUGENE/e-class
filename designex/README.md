# E-Class 디자인 예시 모음

실제 서비스 수준의 UI/UX 디자인 레퍼런스 페이지들입니다.

## 참고 플랫폼

### 한국
- **클래스101** - 감성적이고 따뜻한 디자인, 취미/라이프스타일 중심
- **인프런** - 깔끔하고 정보 중심적, 개발자 친화적
- **패스트캠퍼스** - 프로페셔널하고 신뢰감 있는 디자인

### 글로벌
- **Coursera** - 학술적이고 권위있는 느낌, 화이트스페이스 활용
- **Udemy** - 마켓플레이스 스타일, 다양한 콘텐츠 노출
- **Skillshare** - 크리에이티브하고 모던한 디자인

## 디자인 파일 목록

| 파일 | 설명 | 스타일 |
|------|------|--------|
| `style-a-modern/` | 모던 & 미니멀 | 인프런/Coursera 스타일 |
| `style-b-warm/` | 따뜻한 감성 | 클래스101 스타일 |
| `style-c-professional/` | 프로페셔널 | 패스트캠퍼스 스타일 |

## 실행 방법

```bash
# 프로젝트 루트에서
npm run dev

# 브라우저에서 접속
http://localhost:3000/designex/style-a
http://localhost:3000/designex/style-b
http://localhost:3000/designex/style-c
```

## 디자인 원칙

### 1. 헤더
- 로고 (좌측)
- 메인 네비게이션 (중앙 또는 좌측)
- 검색바
- 사용자 메뉴 (우측: 로그인/크레딧/프로필)

### 2. 히어로 섹션
- 강렬한 메인 카피
- 서브 카피 (가치 제안)
- CTA 버튼 (대비되는 색상)
- 배경 이미지 또는 일러스트

### 3. 코스 카드
- 썸네일 (16:9 또는 4:3)
- 카테고리 뱃지
- 제목 (2줄 제한)
- 강사 정보
- 평점 & 수강생 수
- 가격

### 4. 푸터
- 회사 정보
- 서비스 링크
- 고객 지원
- SNS 링크
- 저작권
